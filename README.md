Why I made this?
================

To be honest, for no reason.
I felt like making a simple UI for latex projects as overleaf is expensive. 
And I love vim, but my collaborators might not. So this is my attempt to solve that problem.

To use this repo
=================

- Prerequisites: 

  - nodejs
  - git
  - pdflatex
  - and a latex distribution like texline or miktex


First install the node packages and then
Run node server.js

1. create a basic github repo for your latex project
2. clone it  by pasting the url to the UI
3. select the project folder from the UI
4. edit, compile and push the changes

I am still working on how to avoid conflict for multiple users
Of course, One way to avoid conflict is to work on different branches
